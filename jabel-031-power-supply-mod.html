JABEL 031 Modified Power Supply

Overview

This repository contains comprehensive documentation and design files for a modified JABEL 031 power supply unit. This is an electronics project that includes PCB layouts, schematics, component lists, and AutoCAD drawings for the external enclosure design.

Created: June 25, 2021

 

Project Description

This project is dedicated to the design and modification of a JABEL 031 power supply unit. The modifications include:

  • Custom PCB Design - Multi-layer circuit board layout and component placement
  • Schematic Diagrams - Complete electrical circuit documentation
  • 3D Enclosure Design - AutoCAD drawings for the external housing
  • Component Documentation - Detailed parts lists and specifications

The project combines both electrical and mechanical design aspects to create a fully documented and reproducible power supply system.

Hardware Design

Electrical Design

The power supply consists of:

  1. Input Stage - Main power conditioning
  2. Transformation Stage - Voltage regulation and isolation
  3. Output Stage - Final power delivery and filtering
  4. Protection Stage - Over-current and thermal protection

Key Specifications:

  • Custom multi-layer PCB design
  • Multiple voltage output rails
  • Protection circuitry for safe operation
  • Optimized component placement for thermal management

Mechanical Design

The enclosure is designed using AutoCAD with:

  • Front Panel Design - User interface and connector placement
  • Housing Structure - Aluminum or plastic enclosure
  • Thermal Management - Ventilation and heat dissipation provisions

Design Files:

  • Panel_przedni_AUTOCAD.dwg - Original front panel design
  • Panel_przedni2.dwg - Alternative/updated front panel variant

Component List

A comprehensive list of all electronic components required for this project is documented in:

  • LISTA_ELEMENTÓW.docx - Editable component list with part numbers
  • PDF/LISTA_CZĘŚCI.pdf - PDF version of the components list

The list includes:

  • Resistors and capacitors (values and tolerances)
  • Active components (ICs, transistors, diodes)
  • Inductors and transformers
  • Connectors and mechanical parts
  • Specifications and reference designators
